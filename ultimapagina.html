OPINION SOBRE EL CURSO

El aprendizaje en este curso fue de gran ayuda, ya que aprendimos muchas cosas que lamentablemente no pudimos aprender dentro o durante el semestre, la verdad es que valio completa y totalmente la pena ya que unas de las muchas cosas que aprendimos fueron:

  1. Aprendi a como crear codigos para:
    1. Crear diferentes tipos de listas de datos
    2. Como cambiar y/o controlar el tamaño de mi tipo de fuente
    3. Cambiar el color de mi texto
    4. Cambiar el tipo de letra
    5. Para cambiar la alineacion de mis parrafos
    6. Para hacer el uso de lineas, de diferente color y grosor
    7. Colocar imagenes de diferentes tamaños, y con un tipo de grosor
    8. Aprendi a poner links para adquirir mas informacion de otra pagina
    9. Para hacer cuestionarios con, diferentes cuadros de texto con diferentes caracteristicas, con botones y casillas de verificacion(varios tipos), con cuadros de listas y listas despegables, areas de texto, etc.

    Aprendimos que al juntar todas estas cosas puedo crear una pagina web y muchas otras cosa, con encuentas, tablas, lineas, letras, colores, imagenes, vovimiento, videos y links de otras paginas.
